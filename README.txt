Adds menu items to MotionBuilder to start listening to the Arduino Remote so it can control motion builder.

INSTALL INSTRUCTIONS

Copy "Remote" into "C:\Program Files\Autodesk\MotionBuilder 2014\bin\x64\python\site-packages"
Copy "arduinoremote.py" into "C:\Program Files\Autodesk\MotionBuilder 2014\bin\config\PythonStartup"

Update the file paths to match your install location and version of MotionBuilder.
